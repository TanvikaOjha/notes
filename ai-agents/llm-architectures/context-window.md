# What is a context window?
The context window of a LLM is the amount of text, in tokens, that a model can consider or "remember" at any one time. A larger context window enables an AI model to process longer inputs and incorporate a greater amount of info into each output.It has a finite length per LLM

context window contains system/developer instructions + retrieved content + tool outputs + chat history

Efficient tokenization can help increase the actual amount of text that fits within the confines of a context window.
